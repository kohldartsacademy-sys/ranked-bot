Monatsranking Vorschlag
=======================

Ziel
----
Das Monatsranking soll nicht mehr direkt nach Elo-Gewinn sortieren.
Starke Spieler sollen fuer Siege weiterhin fair Punkte bekommen, auch wenn sie wegen ihrer hohen Elo nur wenig Elo gewinnen.
Underdog-Siege sollen trotzdem extra belohnt werden.


Punktevergabe
-------------
Nur Siege geben Punkte.
Niederlagen geben 0 Punkte.

Formel:

Wenn der Spieler verliert:
    Monatspunkte = 0

Wenn der Spieler gewinnt:
    Monatspunkte = 10 + Gegner-Bonus + Underdog-Bonus


Gegner-Bonus
------------
Der Gegner-Bonus belohnt Siege gegen hoch geratete Gegner.

Berechnung:
    Gegner-Bonus = (Gegner-Elo - 1000) // 75

Regeln:
    Minimum: 0 Punkte
    Maximum: 6 Punkte

Beispiele:
    Sieg gegen 700 Elo  -> +0 Gegner-Bonus
    Sieg gegen 1000 Elo -> +0 Gegner-Bonus
    Sieg gegen 1150 Elo -> +2 Gegner-Bonus
    Sieg gegen 1300 Elo -> +4 Gegner-Bonus
    Sieg gegen 1450 Elo -> +6 Gegner-Bonus


Underdog-Bonus
--------------
Der Underdog-Bonus belohnt Siege gegen einen Spieler mit hoeherer Elo.

Berechnung:
    Underdog-Bonus = (Gegner-Elo - Eigene-Elo) // 75

Regeln:
    Nur wenn Gegner-Elo hoeher ist als eigene Elo
    Minimum: 0 Punkte
    Maximum: 6 Punkte

Beispiele:
    1400 gewinnt gegen 700:
        Basis: 10
        Gegner-Bonus: 0
        Underdog-Bonus: 0
        Gesamt: 10 Punkte

    700 gewinnt gegen 1400:
        Basis: 10
        Gegner-Bonus: 5
        Underdog-Bonus: 6
        Gesamt: 21 Punkte


Sortierung des Monatsrankings
-----------------------------
1. Monatspunkte absteigend
2. Siege absteigend
3. Winrate absteigend
4. Anzahl gespielter Matches absteigend
5. Elo-Gewinn im Monat absteigend


Kurzformel
----------
if not won:
    monthly_points = 0
else:
    base_points = 10
    opponent_bonus = max(0, min(6, (opponent_rating - 1000) // 75))
    underdog_bonus = max(0, min(6, (opponent_rating - player_rating) // 75))
    monthly_points = base_points + opponent_bonus + underdog_bonus
